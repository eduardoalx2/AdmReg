====================================================
HISTÓRICO DO PROJETO - REGIÃO 655 (MIGRAÇÃO FIREBASE)
====================================================

OBJETIVO:
Unificar todos os minissistemas (Supabase, Google Sheets, Apps Script) em um único banco de dados centralizado usando o Firebase (Firestore) e, futuramente, transformar tudo em um App SaaS com React + Vite.

O QUE JÁ FOI FEITO:
1. Criação de um novo projeto no Firebase ("Regiao655-Admin") com o Firestore Database ativado e configurado.
2. Telas de Gestão (`s_pastores.html` e `s_igrejas.html`) concluídas:
   - Sincronização cruzada: Alterar o titular na tela de Igrejas atualiza o Pastor e vice-versa.
   - Suporte a múltiplas igrejas para o mesmo pastor (ex: Sede Regional e Região Eclesiástica).
   - Filtros dinâmicos e caixas de seleção com busca interna para facilitar a usabilidade.
3. Criação do Painel Central (`index.html`) interligando todos os minissistemas administrativos.
4. Lançador de Saldo (`saldodeviradareg655.html`) atualizado:
   - Puxando a lista de Igrejas e seus respectivos Pastores Titulares dinamicamente do Firebase.
   - Salvando os dados na coleção `saldos_virada` no Firebase (e mantendo paralelamente na planilha do Google por segurança).
5. Portal principal (`index.html`) atualizado com os novos atalhos para o Painel Admin e Sistema de Recibos.

ONDE PARAMOS / PRÓXIMOS PASSOS:
Nós paramos exatamente no início do Passo 2 da migração.

1. (PASSO ATUAL) Atualizar o sistema de Presença (`presencareg655.html`) para puxar a lista de igrejas e pastores dinamicamente do Firestore (atualmente ele ainda usa as tags <option> fixas no HTML).
2. Migrar o banco de dados da Agenda e do Ato Mobiliário do Supabase para o Firebase.
3. Iniciar a estrutura do novo App SaaS em React (Vite) para consolidar todos esses sistemas HTML em uma única aplicação moderna.

COMO RETOMAR O TRABALHO:
Por favor, leia a estrutura atual acima e vamos iniciar o Passo 1 deste novo ciclo: dinamizar os campos de seleção do arquivo "presencareg655.html" usando o Firebase.
